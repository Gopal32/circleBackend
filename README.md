#### To Run This Application Follow The Steps Below : 
```sh
$ git clone 
```
```sh 
$ cd ./demo
```
>Create a file named `.env` in app_root/config folder and paste the content at the end of this document in newly created file.

```sh
$ npm i
```
```sh
$ npm run develop
```
